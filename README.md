# Exercise Question Bank

A collection of questions from Pourya's lecture notes for the course MATB24H3, compiled into one .tex for easy reference and collaborative improvement.

## About

This repository contains practice questions and exercises from lecture materials, which are to be submitted along with the final exam for bonus points. It's designed to help students by having all the questions in one organised location.

Please do consider contributing and adding new questions as they roll out. It is helpful to everyone in the course.
## How to Use

1. **Clone or Fork** the repository
2. **Open the file** and work through the questions
3. **Submit improvements** via pull requests (see Contributing below)

## Contributing

We welcome contributions from anyone! Whether you're:
- Adding new questions as they roll out
- Clarifying existing ones
- Fixing typos or formatting

Just follow these steps:

1. **Fork** the repository
2. Create a **new branch** for your changes (`git checkout -b your-feature`)
3. Make your edits
4. **Commit** with a clear message (`git commit -m "Add questions from section X"`)
5. **Push** and submit a **pull request**
